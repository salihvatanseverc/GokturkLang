# GokturkLang Studio

GokturkLang Studio, GokturkLang programlama dili için geliştirilmiş Electron tabanlı bir geliştirme ortamıdır.

## Özellikler

- GokturkLang yorumlayıcı (interpreter)
- Değişken sistemi
- Matematiksel işlemler (+, -, *, /)
- Karekök işlemi (kok())
- yazdır() fonksiyonu
- AST tabanlı çalışma yapısı
- Yerleşik terminal çıktısı
- .gt dosya desteği (planlanan)
