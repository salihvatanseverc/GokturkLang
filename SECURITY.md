# Security Policy

## Supported Versions

We actively support the following versions of GokturkLang with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :white_check_mark: |
| 0.0.x   | :x:                |
| < 0.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in GokturkLang, please report it responsibly.

You can report vulnerabilities via:

- GitHub Issues (for non-sensitive reports)
- Email: gokturkcom.bg@gmail.com (for sensitive security issues)

### What to expect:
- You will receive an initial response within 48–72 hours.
- We will investigate and validate the issue.
- If confirmed, a fix will be planned in the next patch release.
- You will be informed when the issue is resolved.

We kindly ask that vulnerabilities are not disclosed publicly until a fix has been released.

Thank you for helping improve GokturkLang security.
